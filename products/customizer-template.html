Sticker Product Customizer Template

Implementation Guide

1. Add to <head>


<!-- Firebase Sticker Customizer -->
<script src="https://www.gstatic.com/firebasejs/9.23.0/firebase-app.js"></script>
<script src="../assets/firebase-sticker-customizer.js"></script>
<link rel="stylesheet" href="../assets/firebase-sticker-customizer.css">

2. Add to end of <body>


<script src="../assets/product-customizer-integration.js"></script>

3. How it works

The integration script automatically detects if the current page is a sticker product based on the product title or URL path. If it is, the script will replace the product media with the Firebase sticker customizer iframe.

4. Configuration

You can customize which products get the customizer by editing the isCustomizableProduct check in the product-customizer-integration.js file.
